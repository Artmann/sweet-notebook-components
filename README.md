# Sweet Components

A Streamlit-like component library for Notebooks.

Show a text:

```python
from sweet_notebook_components import st


st.text("Hello, World!")

```

## Development

Activate the environment:

On Windows:

```bash
venv\Scripts\activate

```

On macOS/Linux:

```bash
source venv/bin/activate

```

Run the test suite:

```sh
pytest


```
